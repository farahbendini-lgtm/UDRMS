# University Dormitory & Restaurant Management System (UDRMS)

## Description
This project is a C++ application developed for ENSIA 2025-2026.

The system manages:
- Students
- Dormitories
- Rooms
- Restaurant menus

## Features
- Add and remove students
- Manage dormitories
- Manage rooms
- Prevent room over-occupancy
- Restaurant menu management
- Save and load data from files

## OOP Concepts Used
- Encapsulation
- Inheritance
- Composition
- Exception Handling

## Classes
- Person
- Student
- Room
- Restaurant
- Dormitory
- University

## Technologies
- C++
- Object-Oriented Programming (OOP)
- Qt (GUI)
